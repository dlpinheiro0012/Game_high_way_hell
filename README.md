# Game_high_way_hell
The game is based on Street Fight, but without an dinamic enemy. We are proposing to the player to be the
"faster cliker" and reach the goal in less time.

# Compilação do Jogo
Instalar o Raylib no seu PC: https://www.raylib.com/  
Apertar "CTRL + SHIFT + B".

# Run do jogo
Digitar ".\build\jogo.exe" ou abrir diretamente o executável.
